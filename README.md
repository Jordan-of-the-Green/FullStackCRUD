# ReadMe

# Description:  
FullStackCRUD is an ASP.NET Core MVC application built with .NET 7. It demonstrates a full CRUD workflow using a simple Product model, Entity Framework Core, and Razor views. Ideal for learning full-stack C# development or showcasing skills.

# How to Install and Run the Project:  
ASP.NET Core MVC (.NET 7) Project Code ~  
Step 1
* Ensure your computer can run Visual Studio 2022.
* Ensure enough space to install the IDE and project files.
* Check that your system meets .NET 7 requirements.

Step 2
* Install Visual Studio 2022 (Community, Professional, or Enterprise).
* Include the ASP.NET and web development workload.
* Visit:
          * Visual Studio: https://visualstudio.microsoft.com/
          * .NET SDK: https://dotnet.microsoft.com/
         

Step 3
* Read and accept the terms and conditions for installing Visual Studio and .NET SDK.

Step 4
* Clone or download the project from GitHub.
* Open the solution (.sln) in Visual Studio.
* Restore NuGet packages if prompted.

Step 5
* Run the project by pressing Ctrl + F5 or clicking the Run button.
* The default route opens the Product list page where you can test CRUD operations.

# Key Features:  
• Create, view, edit, and delete Product records  
• Clean MVC project structure  
• SQLite database with EF Core 
• Simple responsive styling with table and form layouts  
• Ready to extend for portfolio or demo projects
